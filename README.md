# WorldTravelMap

Dashboard of the world map with visited places.


## Python

### Install packages local environment

E.g.:
```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
```

Generate requirements:
```pip3 freeze > requirements.txt ```
